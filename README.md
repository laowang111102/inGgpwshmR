# 项目概述 python933

本项目是一款基于 Spring Boot 和 Vue 的酒店管理系统，提供详尽的文档支持。

# 技术栈

- 后端：Spring Boot
- 前端：Vue
- 数据库：MySQL

# 运行环境

- Java版本：JDK 1.8
- Node版本：13.x
- Python版本：2.7
- 集成开发环境：IDEA 或 Eclipse
- 数据库：MySQL 5.x 或 8.x

# 系统角色

系统包括以下三种角色：

- 用户
- 酒店管理员
- 管理员

## 用户功能

- 用户注册/登录
- 在线浏览客房（分类、搜索）
- 预定客房
- 入住客房
- 退房功能
- 评价功能
- 查看公告

## 酒店管理员功能

- 预定信息审核
- 客房入住管理
- 退房信息审核
- 客房管理（增删改查客房信息）

## 管理员功能

- 用户管理
- 酒店管理员管理
- 客房管理
- 预定信息管理
- 客房入住管理
- 退房信息管理
- 评价管理
- 轮播图管理
- 酒店公告管理
